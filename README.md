# 🚀 Flask Admin Dashboard

A dynamic and responsive **Admin Dashboard** built using **Python (Flask)**, designed to manage and visualize data efficiently with full CRUD functionality and interactive charts.

---

## 📌 Project Overview

This project is a backend-driven admin panel where users can:

* Add, update, delete, and fetch data
* Visualize data using charts
* Monitor records dynamically in real-time

It demonstrates practical implementation of **Flask, REST concepts, and frontend integration**.

## 🚀 Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/trinathprasad/flask-admin-dashboard.git
cd flask-admin-dashboard
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
python run.py
```
